## 🚀 Hướng Dẫn Sử Dụng

### Xem Trực Tiếp

Chỉ cần mở file `index.html` trong trình duyệt:

```bash
# Mở trực tiếp
open index.html
```

### Chạy với Live Server (Khuyến nghị)

**VS Code:**
1. Cài đặt extension **Live Server**
2. Right-click `index.html` → **Open with Live Server**
3. Portfolio sẽ tự động mở tại `http://localhost:5500`

### Deploy lên GitHub Pages

```bash
# 1. Push code lên GitHub
git add .
git commit -m "Add portfolio"
git push origin main

# 2. Vào Settings → Pages
# 3. Source: Deploy from a branch
# 4. Branch: main / root → Save
# 5. Đợi vài phút để GitHub deploy
```

### Deploy lên Netlify

**Cách 1: Drag & Drop**
1. Truy cập [netlify.com/drop](https://app.netlify.com/drop)
2. Kéo thả folder portfolio vào
3. Nhận URL live ngay lập tức

**Cách 2: Netlify CLI**
```bash
npm install -g netlify-cli
netlify deploy --prod
```

### Deploy lên Vercel

```bash
npm install -g vercel
vercel --prod
```

## 🌟 Features Chi Tiết

### 🎭 Animations & Effects

- **Floating Avatar** - Keyframe animation lên xuống
- **Gradient Blobs** - Background blobs động với blur effect
- **Skill Bars** - Animate on scroll với Intersection Observer
- **Card Hover** - Transform scale và translateY
- **Nav Links** - Underline animation on hover
- **Active Section** - Auto highlight nav khi scroll

### 📱 Responsive Design

```css
/* Mobile First Approach */
- Mobile: < 768px   → Full width layout
- Tablet: 768px+    → 2 columns grid
- Desktop: 1024px+  → 3 columns grid
```

### ⚡ Performance

- ✅ CDN cho Tailwind và Font Awesome
- ✅ Minimal JavaScript (~50 lines)
- ✅ CSS animations sử dụng GPU (transform/opacity)
- ✅ Lazy loading cho images (nếu thêm)
- ✅ Smooth scroll với CSS
